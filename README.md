# 🏢 Web Studio

## 🔗 [Live Demo](https://milenness.github.io/web-studio/)

**Web Studio** is my very first web development project, representing the foundation of my journey into frontend engineering. It is a clean and professional landing page for a digital agency, designed to showcase effective business solutions and services.

## 🚀 Key Features

* **Semantic Structure** — Built with accessibility and SEO in mind using semantic HTML5 tags.
* **Service Showcase** — Detailed sections for business strategies, punctuality, diligence, and technology highlights.
* **Portfolio Gallery** — A curated display of agency projects, including banking, payment, and meditation apps.
* **Interactive Forms** — Includes functional "Order Service" modals and a newsletter subscription footer.
* **Responsive Styling** — Ensures a consistent and professional look across different screen sizes.

## 🛠 Tech Stack

* **HTML5** & **CSS3**
* **Modern Layouts** — Focused on precise positioning and clean spacing.
* **BEM Methodology** — Applied to maintain a modular and understandable code structure from the start.

## 🎨 Professional Highlights

* Successfully implemented **CSS transitions** for interactive elements like buttons and social media icons.
* Focused on **cross-browser compatibility** to ensure a stable user experience.
* Established a solid understanding of **naming conventions** and project file organization.
